Se placer dans ProjetPOO/
Pour la terminal view:
javac view/TerminalView.java
java view/TerminalView (mettre en grand la fenêtre du terminal)
Pour la view globale:
javac MainView.java
java MainView (*)

(*)Commentaire: lors du lancement du menu, il est possible que la fenetre soit toute blanche, il faut effectuer cette commande quelques fois pour que cela affiche correctement le menu et ses éléments.
